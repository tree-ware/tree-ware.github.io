tree-ware

Accelerate your software development!

The Problem

Companies struggle with feature velocity
They use a lot of open-source software to speed up their development
But that alone is NOT sufficient!
Every new functional feature requires glue-logic across all this open-source software
Every new non-functional features requires modifications to all existing functional features
Very hard to ensure all features are implemented consistently
Bad practices in the initial rush-to-market result in slower subsequent releases

The Solution

tree-ware

An end-to-end data-driven open-source framework to accelerate software development!

End-to-end

UIs, APIs, load-balancers, caches, databases, compute-pipelines, devices

Data-driven

System-wide schema specified by developers

Open-source

https://github.com/tree-ware

How?

Uses a system-wide schema to set up & operate infrastructure
Takes care of non-functional features
Developers only need to define a system-wide schema & custom business-logic

Targets existing infrastructure

Databases, queues, compute-pipelines, etc.

Provides other infrastructure

UI, load-balancer, API server, cache, RBAC, feature-flags, etc.
